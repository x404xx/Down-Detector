## **Installation**

**Using** _`poetry`_

```
git clone https://github.com/x404xx/Down-Detector.git
cd Down-Detector
poetry shell
poetry install
```

**Using** _`pip`_

```
git clone https://github.com/x404xx/Down-Detector.git
cd Down-Detector
virtualenv env
env/scripts/activate
pip install -r requirements.txt
```

## Usage Example

```bash
python -m detector 'facebook'
```
